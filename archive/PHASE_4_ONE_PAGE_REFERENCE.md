## **TROUBLESHOOTING**

### **Crop not dragging?**
- Verify crop mode enabled (aspect ratio selected)
- Check corner handles visible
- Try different corners

### **Text not visible?**
- Check text color (not same as background)
- Increase text size
- Check layer list has items

### **Effects not showing?**
- Shadow blur > 0
- Outline width > 0
- Check color values
